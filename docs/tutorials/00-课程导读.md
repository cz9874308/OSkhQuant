## 📖 欢迎来到量化交易的世界！

### 什么是量化交易？

想象一下，你每天盯着股票软件，看着红红绿绿的数字，心里想着：

> "这只股票涨了 5%，要不要买？"  
> "那只昨天买的跌了 3%，是止损还是加仓？"

这种靠"感觉"做决策的方式，我们称之为 **主观交易**。

而 **量化交易** 则完全不同。它的核心思想是：

> 🧠 **用规则代替直觉，用程序代替手动**

比如，你可以设定这样一条规则：

```
如果 5日均线 > 20日均线，那就买入
如果 5日均线 < 20日均线，那就卖出
```

然后让电脑帮你执行。无论股价涨跌，你都不用纠结——**规则说了算**。

### 一个生活化的比喻 🍳

量化交易就像是 **按菜谱做饭**：

| 主观交易 | 量化交易 |
|---------|---------|
| 凭感觉放盐 | 按菜谱放 5 克盐 |
| 火候看心情 | 中火煮 8 分钟 |
| 结果时好时坏 | 每次都差不多 |

菜谱不能保证每道菜都是米其林水准，但至少不会太离谱。量化策略也是一样——它不能保证你赚钱，但能让你的投资决策 **有理有据、可追溯、可优化**。

## 🤔 为什么选择 OSkhQuant？

市面上有很多量化工具，为什么要选 OSkhQuant（看海量化交易系统）呢？

### 1️⃣ 完全免费、开源

没有隐藏收费，没有付费解锁高级功能。代码全部公开，你可以自由学习和修改。

### 2️⃣ 数据和策略都在你电脑上

你的策略代码、历史数据、回测结果，全部存在本地。不用担心"云端平台"跑路或者策略泄露。

### 3️⃣ 图形化界面，小白友好

不需要从命令行敲代码启动。双击打开软件，鼠标点点就能开始回测。

### 4️⃣ Python 生态全支持

可以在策略里使用任何 Python 库——Pandas、NumPy、甚至 TensorFlow 做 AI 选股。

### 5️⃣ 专为 A 股优化

T+1 交易规则、涨跌停限制、印花税计算……这些 A 股特有的规则，OSkhQuant 都帮你处理好了。
